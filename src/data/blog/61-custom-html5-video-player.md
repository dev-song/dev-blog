JavaScript로 HTML Video 요소 다루기

*   currentTime, duration, autoplay, paused, playbackRate, volume 등 동영상의 속성을 설정할 수 있음

미디어 요소가 재생될 때, 멈출 때 코드 실행하기

*   미디어 요소가 재생될 때, 그리고 멈출 때 코드가 실행되도록 Event Listener를 등록할 수 있음
*   play, pause 이벤트에 대해 Event Listener를 등록해주면 됨
