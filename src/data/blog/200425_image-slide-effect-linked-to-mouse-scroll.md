웹페이지 스크롤한 길이 확인하기

*   웹페이지가 수평/수직으로 화면 크기보다 커서 스크롤할 때, Web API를 통해 스크롤한 길이를 확인할 수 있음
*   형태
    *   수평 스크롤: window.scrollX
    *   수직 스크롤: window.scrollY
*   참고자료
    *   [Window.scrollX - Web API | MDN](https://developer.mozilla.org/ko/docs/Web/API/Window/scrollX)
    *   [Window.scrollY - Web API | MDN](https://developer.mozilla.org/ko/docs/Web/API/Window/scrollY)
